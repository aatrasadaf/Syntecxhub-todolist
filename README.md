# To-Do List App

A **simple, interactive To-Do List web application** built with **HTML, CSS, and JavaScript**. This project allows users to manage daily tasks efficiently with a clean interface and persistent storage.

---

## **Features**

- **Add Tasks** – Quickly add new tasks to your list.
- **Edit Tasks** – Edit task content directly.
- **Delete Tasks** – Remove tasks you no longer need.
- **Mark as Completed** – Click the task or tick icon to mark as completed.
- **Visual Indicators**:
  - **Red circle** for incomplete tasks.
  - **Green tick + line-through** for completed tasks.
- **Persistent Storage** – All tasks are saved in the browser using **LocalStorage**.
- **Click-to-complete** – Toggle completion by clicking on the task text or icon.
- **Responsive Design** – Works well on desktop and mobile devices.

---

## **Technologies Used**

- **HTML5** – Structure of the app.
- **CSS3** – Styling, layout, and visual effects.
- **JavaScript (ES6)** – DOM manipulation, event handling, LocalStorage.
